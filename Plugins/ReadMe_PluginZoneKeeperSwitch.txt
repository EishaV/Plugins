###########################################################################
##  Plugin: ZoneKeeper Switch
###########################################################################

Plugin-Datei: PluginZoneKeeperSwitch.cs

Funktion: Dieses Plugin ermöglicht es, das Zone Keeper Feature eines Mähers ein- oder auszuschalten.

Verwendung: Das Zone Keeper Feature kann aus- oder eingeschaltet werden. Der Mäher verwendet dann die Zone Keeper Logik, wenn das Zone Keeper Feature eingeschaltet ist und wenn Startpunkte verwendet werden. Ist das Zone Keeper Feature ausgeschaltet oder werden keine Startpunkte verwendet, so findet die Zone Keeper Logik des Mähers keine Anwendung.
